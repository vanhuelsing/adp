# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the ADP protocol specification or SDK implementations, please report it responsibly.

**Email:** adp@apideals.ai

### What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response timeline

- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 5 business days
- **Fix timeline:** Depends on severity, typically within 30 days

### Scope

- ADP protocol specification (`spec/`)
- TypeScript SDK (`sdk/typescript/`)
- Python SDK (`sdk/python/`)
- JSON Schemas (`spec/*/schemas/`)

### Out of scope

- Third-party implementations of ADP
- Social engineering attacks
- DoS attacks against infrastructure

## Supported Versions

| Version | Supported |
|---------|-----------|
| v0.2.x  | ✅        |
| v0.1.x  | ✅        |
| < v0.1  | ❌        |
